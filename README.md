## How to Compile

### From a terminal run

```
git clone https://github.com/Manik5/sales-taxes-exercise
```

### Change the working directory with
```
cd sales-taxes-exercise
```

## How to Run

### inside the sales-problems directory, run:
```
ruby run.rb  "nameofthefile".txt
```
#### example
```
ruby run.rb cart1.txt
```
